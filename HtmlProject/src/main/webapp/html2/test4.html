img src 99쪽 이미지 삽입하기 태그


웹에서 사용하는 이미지 형식 : jpg, png, gif

-jpg : 사진 형태, 색상, 명암을 다양하게 표현
-gif : 256색의 색상 사용. 그래픽 프로그램으로 그린 로고 등 작은 이미지 표현
-png : 다양한 색상 표현, 투명한 배경색, 웹에서 많이 사용

html 파일과 같은 폴더에 있는 사진만 불러올 수 있음.
-픽셀 : 화면을 이루는 빛(하나의 점), 해상도
-px : 이미지 크기를 픽셀로 지정하고 단위는 무조건 px로 고정
-% : 이미지 크기를 브라우저 창의 가로 길이를 100%로 보고 비율로 크기를 조정하는태그
